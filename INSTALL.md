# Nulla Extension — Installation Guide

## How to install in Chrome (Developer Mode)

### Step 1 — Download the release
Download `nulla-extension-v0.62.6.zip` from the [GitHub Releases](../../releases) page.

### Step 2 — Unzip
Extract the zip to a folder on your computer, e.g. `nulla-extension-v0.62.6/`.

### Step 3 — Open Chrome Extensions page
In Chrome, navigate to:
```
chrome://extensions
```

### Step 4 — Enable Developer Mode
In the top-right corner of the Extensions page, toggle **Developer mode** ON.

### Step 5 — Load the extension
Click **"Load unpacked"** and select the folder you extracted in Step 2 (the one that contains `manifest.json`).

### Step 6 — Done
The **Nulla Extension** icon will appear in your Chrome toolbar. Pin it via the puzzle icon (🧩) if it is not immediately visible.

---

> **Note:** Chrome may show a warning that the extension is not from the Chrome Web Store — this is expected for manually loaded extensions.
> You may need to re-enable it after each Chrome update if Chrome disables it automatically.
